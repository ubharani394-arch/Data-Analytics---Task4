# Task 4 – Data Storytelling & Statistical Validation

## Project Overview
This project analyzes a sales dataset using Python to identify business insights through data visualization and basic statistical analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- Sales Dataset
- Features include:
  - Product
  - Category
  - City
  - Customer Name
  - Order Date
  - Quantity
  - Unit Price
  - Total Sales

## Charts Created
1. Bar Chart – Total Sales by Category
2. Line Chart – Monthly Sales Trend
3. Pie Chart – Sales Distribution by Category
4. Histogram – Quantity Distribution
5. Box Plot – Total Sales Distribution
6. Scatter Plot – Unit Price vs Total Sales

## Key Insights
- Electronics generated the highest sales.
- Sales varied across different months.
- Product categories contributed differently to total revenue.
- Most orders had moderate quantities.
- A positive relationship exists between Unit Price and Total Sales.

The project demonstrates how Python can be used for data storytelling by creating meaningful visualizations and extracting business insights from sales data.
